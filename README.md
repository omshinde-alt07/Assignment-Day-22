# Assignment-Day-22

## 📁 Files Included

* `am.ipynb` → Contains AM assignment (Pandas, data analysis, visualization)
* `pm.ipynb` → Contains PM assignment (Machine Learning tasks like regression & classification)

---

## 📌 Dataset Used

The dataset used in this assignment is the **Iris dataset**.

* It contains information about flower measurements:

  * Sepal length
  * Sepal width
  * Petal length
  * Petal width
* Target variable:

  * Species (setosa, versicolor, virginica)

The dataset is loaded using **sklearn**, so no external download is required.
